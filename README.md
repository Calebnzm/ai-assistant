# My AI powered Personal Assistant

This is a personalized AI assistant to help me with everyday tasks, helping to streamline my tasks, projects, goals, and habits.

Functionalities:
    **Task Management**
        - This is the most basic functionality of the application/assitant. It allows me to keep track of my habits, as well as project and tasks.
          I can add, modify and delete tasks, habits and projects. I get a list of all pending and impending tasks everyday, that I can keep track of.
    **AI Guidance**
        - The application is integrated with an LLM, to provide basic guidance, tips as well as assistant. This LLM will be integrated with MCP servers to 
          provide access to tools such as Google Workspace tools.


    
More Features:
The application only has the two aforement features but the following features will be added:
    **Email monitoring**
        - The assistant will be able to monitor your email, taking all the actions neccessary.
    **Google & Microsoft Workspace Integration**
        - The assistant will be augmented with tools to interact with all your google, as well as microsoft services
    **Social Media Integration**
        - The assistant will be accessible through social media such as WhatsApp, Telegram, etc, giving it the full feel and
          capabilities of a complete personal assistant, making it easily accessible and usable.
